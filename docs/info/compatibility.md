# 🔗Compatibility

## **Direct compatibility**

### <mark style="color:red;">Directly</mark> supported item plugins list

* ItemsAdder
* Oraxen
* EcoItems
* EcoArmor
* MMOItems
* MythicMobs
* eco
* NeigeItems
* ExecutableItems
* Nexo
* CraftEngine

You can use ItemBridge as custom item provider which supports more custom item plugins, click [here](https://ultimateshop.superiormc.cn/format/itembridge) to know more.

### <mark style="color:red;">Directly</mark> supported protection plugins list  <a href="#directly-supported-protection-plugins-list-premium" id="directly-supported-protection-plugins-list-premium"></a>

If players do not have permission to break block within these protection plugins areas, UltimateTweak can prevent players use some tweaks in these areas.

* BentoBox
* Dominion
* GriefPrevention
* HuskTowns
* HuskClaims
* Lands
* PlotSquared
* Residence
* Towny
* WorldGuard
* SuperiorSkyblock2

### <mark style="color:red;">Directly</mark> supported block plugins list <a href="#directly-supported-block-plugins-list" id="directly-supported-block-plugins-list"></a>

By being compatible with these plugins, UltimateTweak can recognize blocks from these plugins and support custom blocks from those plugins is some tweaks.

{% hint style="warning" %}
<mark style="color:red;background-color:red;">**ONLY**</mark> custom block from **CraftEngine** is <mark style="color:red;">offical support</mark>.

Due to the fact that the custom blocks of other plugins have not truly registered new blocks at the server-side level, although we have tried to add some compatibility code, we cannot guarantee that it will be 100% useful on this and **will not make any fixes**. We welcome community users to try to further improve compatibility code, but don't expect us!
{% endhint %}

* ItemsAdder
* Oraxen
* MMOItems
* CraftEngine
* Nexo
