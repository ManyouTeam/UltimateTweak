# Welcome to **UltimateTweak**

> **UltimateTweak** is a modular quality-of-life plugin that brings useful survival tweaks to Spigot, Paper, and Folia servers.

---

## Built-in Tweaks

| Tweak | Description |
| :-- | :-- |
| **Best Tool** | Automatically switches to the best available tool when breaking a block. |
| **Tree Cutter** | Cuts down an entire tree at once with configurable falling animations and effects. |
| **Tree Replant** | Automatically replants dropped saplings on valid soil. |
| **Vein Mine** | Mines connected ore veins with configurable limits and search rules. |
| **Dynamic Light** | Creates dynamic light around players based on held items. |
| **Entity Vehicle Restriction** | Prevents matched entities from entering boats or minecarts. |

---

## Flexible Configuration

Every tweak has its own configuration file and can be enabled, disabled, or customized independently.

UltimateTweak also provides flexible systems for:

- Item matching
- Entity matching
- Conditions and actions
- Custom items and blocks
- Protection plugin compatibility

---

## Plugin Compatibility

UltimateTweak supports many popular custom item plugins, including:

- ItemsAdder
- Oraxen
- Nexo
- MMOItems
- EcoItems and EcoArmor
- MythicMobs
- ExecutableItems
- CraftEngine
- NeigeItems

It also supports many protection plugins, including:

- WorldGuard
- Residence
- Towny
- Lands
- PlotSquared
- GriefPrevention
- HuskClaims and HuskTowns
- Dominion
- SuperiorSkyblock2
- BentoBox

## Server Support

- Spigot
